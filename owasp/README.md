# OWASP Foundation Contributions

> A collection of vulnerability assessment protocols, testing methodologies, and framework architecture contributed upstream to the OWASP Foundation.

---

## Overview

This directory contains my open-source contributions to the official **OWASP Web Security Testing Guide (WSTG)** upstream repository. The focus of these projects is modernizing legacy security documentation, drafting explicit API testing methodologies for enterprise environments, and ensuring strict CI/CD pipeline compliance against global WSTG style guidelines.

---

## Key Highlights

- **01-api-security-testing:** A comprehensive migration and rewrite of REST API testing methodologies, focusing on OpenAPI reconnaissance, Mass Assignment, and HTTP Verb Tampering within the modular WSTG framework.
